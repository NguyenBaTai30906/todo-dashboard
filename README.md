# Todo Dashboard

A task management app built with Next.js, TypeScript, and TailwindCSS.

## Features

- Add, edit, delete tasks
- Filter by status (pending/completed) and priority (high/medium/low)
- Real-time search
- Data saved to localStorage
- Dark theme, responsive design

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Custom Hooks (useTasks, useTaskFilter, useTaskStats)

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Project Structure

```
src/
├── app/          # Pages and layouts
├── components/   # UI components
├── hooks/        # Custom hooks
└── types/        # TypeScript types
```
